NPGCSystem

ไม่มีขีดจำกัดในการใช้งานของตัวรันนี้ และดาวน์โหลดฟรีไม่เสียเงิน ฟังชั่นไม่ซ้ำใคร มีฟีเจอร์เยอะ

ปลั๊กอิน

เป็นตัวช่วยของเซิร์ฟ และยังสามารถรัน api เวอร์ชั่นเก่าได้ นี้คือฟังชั่นของตัวรันเซิร์ฟนี้

โลก

สามารถรองรับได้หลายโลก หรือสามารถโหลดมาลงได้เลย ไม่ต้องสร้างให้เสียเวลา

จัดการไฟล์

จัดการง่ายๆ และแก้ไขง่ายๆ ไม่เสียเวลาในการจัดการไฟล์ต่างๆ

เซิร์ฟเวอร์

เปิดได้ไม่จำกัด และยังมีการตั้งค่าจำนวนผู้เล่น มากหรือน้อย ตามใจคนใช้งานเลย

ไม่มีขีดจำกัดในการควบคุม

ควบคุมได้สบายทั้งในเกมและนอกเกม เป็นฟังชั่นของทุกตัวรันในเซิร์ฟมายคราฟ

มากมายหลากหลายฟังชั่น

มีหลากหลายฟีเจอร์ และยังเป็นตัวรันเวอร์ชั่นล่าสุดที่รองรับมายคราฟพีอีเวอร์ชั่นปัจจุบัน
